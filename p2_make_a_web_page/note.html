What is HTML?

HTML is HyperText Markup Language which is the protocol between the Internet and servers.

HTML Markup

Element:

<name> contents </name>

In this case, <name> is an opening tag and </name> is a closing tag. Don't forget to write closing tag.

More tags

<bold> defines bold
<em> defines emphasis
<a> refers to a link
Some tags such as <br> have no closing tag which are called void tag.You can also insert images . Also, we can sepcify color by using a span tag.

We can also specify a block style by using a div tag.

CSS Vocabulary

  • CSS: Cascading Style Sheets
  • Rule: A line of CSS code describing the value that a certain attribute should take.
  • Property: The property you want to change.
  • Value: The value that you want to assign to the attribute.
  • Selector: The name that you use to in order to target the elements that are assigned to a class or id attribute in the HTML.

Why CSS is important?

  1. Save time.
  2. Write less code.
  3. Prevent organizing mistakes.
