What is HTML?

HTML is HyperText Markup Language which is the protocol between the Internet and servers.

HTML Markup

Element:

‹name› contents ‹⁄name›

In this case, ‹name› is opening tag and ‹⁄name› is closing tag. Don't forget to write closing tag.

More tags

‹bold› defines bold
‹em› defines emphasis
‹a› refers to a link
Some tags such as ‹br› have no closing tag which are called void tag.You can also insert images . Also, we can sepcify color by using a span tag.

We can also specify a block style by using a div tag.
